# Yazi

Installed with `make install`: Homebrew `yazi`, `fd`, `ripgrep`; Catppuccin Mocha flavor copied under `~/.config/yazi/flavors/`; `yazi.toml` Stow’d from `yazi/.config/yazi/` in this repo.


| Command | Action                                                                                      |
| ------- | ------------------------------------------------------------------------------------------- |
| `y`     | Open Yazi; on **q**, shell `cd`s to the last folder; **Q** quits without changing directory |
| `yazi`  | Same binary, no cwd handoff (use `y` in normal workflows)                                   |


See the in-app help (F1 / `~`) and [keybindings](https://yazi-rs.github.io/docs/quick-start#keybindings).
